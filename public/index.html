📖 Panduan Penggunaan Bot

Berikut adalah beberapa panduan penting untuk mengatur dan menggunakan bot Anda.

1. Panduan Koneksi Google Sheets (Rekap CT)

// 1. Buat Spreadsheet baru di Google Sheets.

// 2. Buat header di baris 1 (A1 - H1) persis seperti ini: waktu, id_pelanggan, nama, tarif, daya, no_gangguan, token, pembuat

// 3. Klik menu Extensions > Apps Script.

// 4. Hapus kode bawaan, dan COPY-PASTE kode sakti di bawah ini:


function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    
    sheet.appendRow([
      data.waktu,
      data.idpel,
      data.nama,
      data.tarif,
      data.daya,
      data.nogan,
      data.token,
      data.pembuat
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({ "status": "sukses" })).setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "pesan": error.message })).setMimeType(ContentService.MimeType.JSON);
  }
}

// 5. Klik Save 💾, lalu klik Deploy > New Deployment.

// 6. Pilih tipe Web App. Bagian 'Who has access' pilih 'Anyone'.

// 7. Klik Deploy. Nanti akan dapat Web App URL panjang.

// 8. Copy URL tersebut dan paste ke kolom GOOGLE_SHEETS_URL di tab Config (.env) web ini!
